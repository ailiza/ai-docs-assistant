# Engineer Onboarding Guide

## Welcome to Spotify Engineering

Your first week will focus on getting your local environment set up and meeting your team.

### Day 1 Checklist

- Get your laptop set up using the IT portal at it.spotify.com
- Clone the main monorepo: `git clone github.com/spotify/backend`
- Install spotctl: `brew install spotctl`
- Join your team Slack channels

### Getting Access

All access requests go through the Access Portal at access.spotify.com. Common things you'll need:
- GitHub org access
- AWS console access
- Datadog access for monitoring
- PagerDuty for on-call

### Your First Ticket

New engineers are assigned a "starter ticket" in Jira. These are small, well-scoped tasks designed to get you familiar with the codebase. Ask your tech lead if you haven't received one by day 2.
